  1. Preparation Steps:
    • Confirm site access and NOC contact information.
    • Confirm all LEDs on laptop NIC function correctly, and disable Wi-Fi to prevent routing conflicts.
    • Physical Layer Validation: Connect a laptop directly to designated switch/device port using a 10GB Ethernet patch cable, and confirm link on laptop and switch (reseat and/or test with different cable if no link light).
  2. Check Public IP by visiting What Is My IP Address?
  3. Check Private IP by running the command ipconfig. Running ipconfig /all will provide more detailed information.
  4. Perform a Ping Test to verify connectivity.
    • ping IP Address: Validate gateway IP reachability.
    • ping 8.8.8.8: Tests basic internet connectivity.
    • ping www.google.com: Initiate a DNS looup and confirm resolution.
